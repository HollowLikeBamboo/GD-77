"gd77_fullflash" contains the MCUXpresso project for debugging and uploading an unmodified full flash readout image.

"gd77_V311_asm_full_flash_readout" contains the MCUXpresso project for full flash readout of FW V3.1.1 using assembler.

"gd77_V311_c_full_flash_readout" contains the MCUXpresso project for full flash readout of FW V3.1.1 using C.

"gd77_V318_asm_full_flash_readout" contains the MCUXpresso project for full flash readout of FW V3.1.8 using assembler.

"gd77_V318_c_full_flash_readout" contains the MCUXpresso project for full flash readout of FW V3.1.8 using C.
